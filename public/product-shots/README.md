# Product Shot Demo Assets

These images are locally bundled Pexels fashion photographs used to validate the
mobile product-selection experience. They are visual stand-ins for the existing
mock catalog records and are not claimed to be photographs of the named Yintai
SKUs.

Production must replace them with approved images from the Yintai product
information system while keeping the same `StoreItem.imageUrl` contract.
