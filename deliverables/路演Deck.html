AI 内容引擎：可控的四层生成

Identity Lock → Garment Lock → Motion/Camera Reference → Validation Gate

🔐 Identity Lock

用户照片 → 面部特征提取 → 多角度角色参考 → 角色库
真实人物：需授权 + 身份验证；不公开原照片

👕 Garment Lock

5层商品图+SKU信息 → 颜色/轮廓/Logo一致性锁定
每层独立一致性检查，低于阈值触发重试

🎬 Motion/Camera Reference

站姿/走秀/转身/打招呼动作参考 → 场景/镜头/表情指令
2–4人共同入镜时保持各自角色与商品

✅ Validation Gate

身份一致性 · 商品一致性 · 时序闪烁 · 穿模 · 解剖 · 审核
mediapipe/vlm 检查 → 逐项打分 → 通过才发布

当前原型为前端效果演示（providerStage=effect-preview）。以上四层控制是一套已存在并可按需接入的生产技术路线（LibTV/Seedream/Seedance/私有GPU），不代表比赛原型已完成商业API联调。
