# AI 喵搭 - OPC 商业挑战赛交付索引

## 核心机制

```text
每日城市角色章节
→ 用户建立动漫身份
→ 选择 5 层完整银泰穿搭
→ AIGC 生成个人角色与内容变体
→ 分享邀请好友创建自己的角色
→ 2–4 人共创互动海报
→ 自愿公开到银泰穿搭广场
→ 看同款 / 商品详情 / 门店任务 / 次日章节
```

体验约 60 秒即可完成，但**不限时、无倒计时**。社交机制不是 A/B 投票或好友裁决，而是每位好友独立创建角色后进入同一场景。

## 评委评分点对应

| 要求 | 项目回答 |
|---|---|
| 每日打开引导 | 7 天连续角色章节；天气、商圈、商品池和收藏卡每日变化 |
| 社交分享动作 | 分享的是“等你入镜”的角色邀请；好友完成自己的身份和 Look 后加入 2–4 人海报 |
| AIGC 实质应用 | AI 生产个人角色、动作/表情变体、多人构图与公开海报，不只做推荐文案 |
| 商业闭环 | 商品图选择 → 详情字段 → 看同款 → 门店任务 → 次日回访 |
| 原型完成度 | 手机 H5 可运行，邀请链接、多人席位与公开广场可现场验证 |

## 真实性边界

- 当前 H5、选装交互、角色编辑状态、邀请链接、2–4 人房间和广场均可运行。
- 角色图片和多人海报明确属于效果演示，生产版由 Gateway AIGC Provider 替换。
- 当前商品图及价格/库存是项目样例数据，正式版接银泰 PIM 与库存服务。
- 私有 GPU 不向浏览器暴露；生产请求只经 API Gateway。
