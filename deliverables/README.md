## 🚀 如何运行原型

```bash
cd ai-meow-outfit
npm install
npm run dev
# 打开 http://localhost:5173
```

核心页面：
- `/` — 首页（天气穿搭推荐）
- `/try-on` — 3D 虚拟试穿
- `/onboarding` — 引导页（身型/风格/门店）
- `/diary` — 穿搭日记
- `/challenges` — PK 挑战 + 邀请
- `/store` — 门店好物
