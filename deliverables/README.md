# AI喵搭：OPC Bounty 03 参赛交付物

## 当前主方案

正式路演主体验已切换为“喵街 AI 今日角色”：

```text
每日角色副本
→ 60 秒三轮选择
→ AI 生成 A 保留版 / B 反转版
→ 好友选择最终封面
→ 解锁银泰到店券和可购同款
```

运行入口：`/#/game`

旧版 3D 人体与 Avatar 页面保留为技术探索，不进入本轮路演主视频。

## 如何运行

```bash
npm install
npm run dev
```

打开 `http://localhost:5173/#/game`。如果端口已占用，以 Vite 输出端口为准。

## 大赛要求对应

| 大赛要求 | 当前方案 |
|---|---|
| 互动 H5 | React + TypeScript 的完整手机交互闭环 |
| 每日打开引导 | 七日连续剧副本、每日场景/商品池、角色卡和次日预告 |
| 社交分享动作 | 发起者分享真实 A/B，好友选择最终封面并获得双方奖励 |
| AIGC 实质应用 | AI 决定 B 方案、评分解释、效果图、文案和商品映射 |
| 商业转化 | 银泰商品、品牌、价格、门店楼层、到店试穿券 |

## 能力边界

- A/B 数据结构、选择逻辑、分享参数、好友投票页面和门店任务已真实实现。
- 当前离线原型使用确定性 `DemoPersonalizedQuestProvider`，不声称已运行大模型。
- 个人穿搭海报当前为前端效果示意；生产目标通过 Gateway 调用私有 AIGC 算力并由 CDN 返回生成图。
- DAU、分享率和 GMV 影响必须通过灰度实验验证，不使用未经验证的确定增长承诺。
