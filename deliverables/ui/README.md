## 能力边界

- 当前 H5 已真实实现每日内容轮换、三轮选择、A/B 候选生成、好友投票链接、奖励解锁与到店任务闭环。
- 当前 `demo-personalized-provider` 是可运行的规则化个性推荐 Provider；它不冒充线上大模型。
- 画面中的穿搭海报是比赛原型效果示意，生产版将由 Gateway 后的 AIGC Provider 生成个性化人物穿搭图。
