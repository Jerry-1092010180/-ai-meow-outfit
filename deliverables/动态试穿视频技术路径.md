# AI喵搭 — 动态试穿/数字人商品展示视频技术路径

## 概述

本文描述从用户授权照片、商品图和动作参考到生成 AI 数字人商品展示视频的完整工程技术路线。**这是一条已存在并可接入的生产技术路线，不代表当前比赛原型已经完成以下所列多模态视频生成商业 API 联调。**

## 能力范围

生产版支持以下视频类型：

| 类型 | 角色风格 | 商品展示 |
|------|----------|----------|
| 写实真人商品展示 | 保持用户原貌 | 5 层穿搭逐层展示，品牌外观可辨认 |
| 漫感/2.5D 电子角色 | 漫画化身份角色 | 穿搭与商品信息与角色资产绑定 |
| 2–4 人共同入镜 | 各自角色保持 | 多角色互动同框，可分别展示穿搭 |

支持的动作指令：站姿展示、走秀、转身、打招呼、指向商品、合拍互动。

每条视频内容绑定 SKU、商品详情页、门店入口和转化埋点。

## 技术架构

```text
用户授权照片
  → Identity Asset（真人或漫感角色）
    → 多角度角色参考 / 角色库
      → 5 层商品图与 SKU 信息
        → 动作参考视频或姿态条件
          → 场景 / 镜头 / 表情指令
            → 多模态视频生成 Provider
              → 身份一致性检查
              → 商品颜色 / 轮廓 / Logo 一致性检查
              → 时序闪烁 / 肢体 / 穿模检查
                → 生成 5–15 秒商品展示短片
                  → 绑定商品详情 / 门店 / 分享房间
```

## 四层生成控制

### 1. Identity Lock（身份锁定）

将用户照片转换为可复用的角色资产，支持写实和漫感两种风格：

- 面部特征提取：脸型、五官比例、发际线、主要辨识特征。
- 多角度角色参考生成：正面、3/4 侧、侧面、背面。
- 角色库管理：同一用户的多个角色版本（不同服装、发型）。
- 肖像授权与撤回机制：用户可随时删除角色资产。

### 2. Garment Lock（商品锁定）

将 5 层商品图的结构、颜色和轮廓锁定到生成内容中：

- 逐层商品输入：内搭、外套、下装、鞋履、配饰。
- 颜色一致性：LAB 色差 ≤ ΔE 3.0。
- 轮廓与结构：领型、袖长、衣长、裙长、Logo 形状。
- Logo 检查：不虚构品牌 Logo，不生成未选择商品。
- 失败处理：单层低于阈值（≤ 0.72）时局部重试。

### 3. Motion / Camera Reference（动作与镜头参考）

接收动作参考视频或姿态序列，控制生成动作：

- 站姿展示（正面、3/4 侧、背面旋转）。
- 走秀（T 台步、猫步，可指定速度）。
- 转身（180°、360° 转身展示）。
- 打招呼（挥手、指向商品）。
- 镜头语言：近景商品特写 → 中景角色展示 → 远景全身。
- 多人场景：2–4 人各自姿态独立控制 + 互动约束。

### 4. Validation Gate（验证门）

生成后的质量控制：

| 检查项 | 方法 | 阈值 |
|--------|------|------|
| identityConsistency | 面部特征比对 | ≥ 0.80 |
| garmentConsistency | 逐层颜色+轮廓比对 | ≥ 0.72/层 |
| temporalConsistency | 帧间光流平滑度 | ≤ 较大帧间跳变 |
| anatomyScore | 肢体关键点完整性 | ≥ 0.85 |
| moderationResult | 内容审核 + 未成年人保护 | 通过/驳回 |

每项检查结果记录到 `traceId`，用于回滚和重试。

## 视觉试穿 vs 真实试穿

本技术路线主要证明的是**视觉试穿 / 数字人商品展示 / 穿搭效果与姿态预览**，包括：

- AI 数字角色穿着选定商品的可视化效果。
- 动态姿态下的穿搭展示。
- 身份、商品与动作的可控生成。

**它不自动证明下列能力**，如需精准尺码需单独建设：

- 真实尺码、松紧度和面料物理准确性。
- 人体参数化建模与服装版型匹配。
- 3D 布料仿真与物理碰撞检测。
- 专业虚拟试穿（VTON）管线验证。

如需要精准尺码试穿能力，应在独立路线图中说明人体参数采集、服装 3D 建模和布料仿真建设。

## 安全性

- 肖像授权与撤回：用户授权照片用于生成，可随时删除角色资产。
- 原照片默认不公开：不进入公开广场或分享内容。
- 真实人物生成需身份验证或明确授权协议。
- AI 生成内容标识：视频和海报添加 AI 生成水印。
- 品牌 Logo 与商品结构幻觉检查：Garment Validation Gate。
- 未成年人保护：检测到未成年人照片时限制生成或提示监护人授权。
- UGC 审核：公开内容经 moderation Gate 检查后发布。
- Provider 失败降级：显式返回 `fallbackReason`，前端不得用模板图冒充个性化结果。
